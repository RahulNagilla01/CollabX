CollabX is a collaboration platform designed for small-medium scaled teams. This platform can be used by teams at Hackathons, Semester project groups etc. I designed this project
by having two main goals, reduction of unnecessary complicated UI and improve functionality. 

Tech stack: 1. HTML, CSS, Javascript(React framework) <br/>
            2. MySQL, Java(Springboot framework) 


<br/>
I made this project to impliment the learning of client-server architecure, Springboot, DBMS taught in the course CSET: CSET109 – Object Oriented Programming using Java, Bennett University.
